Projects and Coursework

Academical Village People Website - Redesigned and deployed a new Django website for the Academical Village People that allowed for content management and e-commerce operations. Switching to the Heroku deployment system also saved the group $150 per year in hosting costs.
Service Learning Practicum - Implemented a data management system for Georgia's Healing House, a non-profit organization in Charlottesville, VA, alongside 5 other senior students. The system utilized the Django framework and other Python libraries for statistical analysis and display.
Mobile Application Development - Developed Move-N-Groove, an Android application that utilized the Spotify API to stream playlists based on the user's environment and movement. This data was taken from the device peripherals and GPS location.
Computer Game Design - Created Blinded, a 2D shooter game built with a fully-featured game engine written in Java. This project was done alongside another student, and it was featured at a games expo where other students and professors played the final product.
